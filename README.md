# Poll Results Visualizer

## Overview
An end-to-end Data Science project that simulates and analyzes poll/survey data using Python and Streamlit. It provides interactive dashboards and business insights based on demographic behavior.

## Problem Statement
Organizations need fast, clear insights from survey data to support decision-making. Manual analysis is slow and inefficient.

## Solution
This project automates:
- Synthetic poll data generation
- Data cleaning and preprocessing
- Exploratory data analysis
- Interactive dashboard creation
- Business insight generation

## Features
- Synthetic dataset generator with realistic bias
- Demographic-based analysis
- Interactive Streamlit dashboard
- Deep analytics mode
- Automated business insights

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Plotly
- Streamlit

## Project Structure
- data/ → dataset
- src/ → core logic
- app/ → Streamlit dashboard
- outputs/ → charts

## How to Run
pip install -r requirements.txt

python src/data_generator.py
python src/visualization.py

streamlit run app/streamlit_app.py
